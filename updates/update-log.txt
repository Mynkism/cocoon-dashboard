PACE LOGIC
  - Sales & Spend: pace based on projected full-month value vs target (linear daily rate)
  - ROI: no projection shown (can't linearise a ratio) — pace based on actual vs target directly
  - On Track band: projected/actual within ±1% of target
  - Ahead: projected/actual more than 1% above target
  - Behind: projected/actual more than 1% below target

DONUT CHART
  - Ring chart fills proportionally to actual vs target (capped at 100%)
  - Spend: always teal (no polarity)
  - Sales/ROI: teal when progressing, amber only if lower_is_better and over target
  - Percentage label centred in ring; shows ✓ when target is hit

CARD LAYOUT

  - Right side: fixed 140×140px donut, flush to card edge with 20px padding
